# Frontend do DVAiS Mentor IA

Aplicação Next.js 14 com App Router que serve como interface web do DVAiS Mentor IA.

## Tecnologias

- **Next.js 14** - Framework React com App Router
- **TypeScript** - Tipagem estática
- **Tailwind CSS** - Estilização
- **TensorFlow.js** - Processamento de IA no cliente
- **PWA** - Progressive Web App (em produção)

## Estrutura

- `app/` - Rotas e páginas (Next App Router)
- `components/` - Componentes React reutilizáveis
- `lib/` - Utilitários e configurações
- `public/` - Assets estáticos
- `types/` - Definições de tipos TypeScript

## Scripts Disponíveis

- `npm run dev` - Inicia servidor de desenvolvimento
- `npm run build` - Gera build de produção
- `npm run start` - Inicia servidor de produção
- `npm run lint` - Executa linter
- `npm run analyze` - Analisa bundle size
