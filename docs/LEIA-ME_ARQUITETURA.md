# Arquitetura do DVAiS Mentor IA

## Objetivo do Projeto

O DVAiS Mentor IA é uma plataforma de mentoria inteligente que utiliza arquitetura em camadas para organizar componentes, serviços e funcionalidades de forma escalável e manutenível.

## Navegação pelos Níveis

A arquitetura segue uma hierarquia de níveis:

- **Corpo** → Estrutura principal e organização geral do projeto
- **Órgãos** → Módulos e serviços principais (apps, services, packages)
- **Células** → Componentes e funcionalidades específicas
- **Átomos** → Elementos básicos e utilitários reutilizáveis

## Painel Web (Frontend)

O frontend do projeto está localizado em:
- `apps/painel-web/` - Aplicação Next.js com interface de usuário
