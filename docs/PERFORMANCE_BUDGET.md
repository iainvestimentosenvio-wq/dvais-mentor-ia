# Performance Budget

## Metricas alvo (desktop)
- **Performance (Lighthouse)**: >= 90
- **Acessibilidade**: >= 90
- **Best Practices**: >= 92
- **SEO**: >= 90

## Web Vitals alvo
- **LCP**: <= 2.5s
- **INP**: <= 200ms
- **CLS**: <= 0.1
- **TTFB**: <= 800ms

## Principios
- Priorizar conteudo acima da dobra.
- Minimizar JS bloqueante no carregamento.
- Evitar animacoes pesadas em dispositivos low-end.
